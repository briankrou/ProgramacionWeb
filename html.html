HTML HyperText Markup Language

HTML (HyperText Markup Language ) Es un lenguaje que esta compuesto por un grupo de etiquetas , que consinte en una etiqueta de inicio y una etiqueta final que es basicamente la misma de inicio pero con una barra "/" , esta se emplea ya sea por que que lo requiere o por buena practica en algunas ocaciones tambien consiten en una sola etiqueta o tambien llamada elementos (por ejemplo <br>), estas etiquetas se le puede adignar atributos acompañados de su valor.

Un ejemplo de estas etiquetas es la etiqueta <html> empleada para que todo lo que este dentro de estas eqtiquetas sea interpretado como código HTML

<html lang="es" > ... </html>
